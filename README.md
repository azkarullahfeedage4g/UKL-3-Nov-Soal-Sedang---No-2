# UKL-3-Nov-Soal-Sedang---No-2
Program ini dibuat untuk menghitung volume tabung berdasarkan jari jari dan tinggi tabung yang diinput oleh pengguna
Program ini menggunakan fungsi method dan parameter untuk memisahkan proses perhitungan dari bagian utama program sehingga lebih rapi dan mudah dipahami.
Cara Kerja Program ini yaitu :
Pertama tama Pengguna diminta untuk memasukkan jari jari dan tinggi tabung melalui Scanner.
Selanjutnya jari jari dan tinggi tersebut dikirim ke fungsi hitung Volume, yang menghitung volume tabung menggunakan rumus:
V = 3.14 x r x r x t
Hasil perhitungan dari fungsi dikembalikan ke program utama (fungsi main) dan disimpan dalam variabel.
Program menampilkan hasil volume tabung ke layar menggunakan System.out.println dalam satuan cm^3
